# Hollee Desktop

![Hollee Desktop](assets/banner.jpg)

**Fall in love with work again.**

Hollee Desktop brings Hollee into a focused desktop app, giving teams a cleaner place to work with the live Hollee platform outside the browser.

## Downloads

- **Windows:** download `Hollee-Desktop-Windows-Setup-x.y.z.exe`.
- **Mac:** download `Hollee-Desktop-Mac-Setup-x.y.z.dmg`.

## Desktop features

- **Global shortcuts:** press `Ctrl+Alt+Space` / `Ctrl+Shift+H` on Windows or `Cmd+Option+Space` / `Cmd+Shift+H` on Mac to show or hide Hollee from anywhere.
- **Tray/menu-bar access:** open Hollee, jump to key pages, or quit from the Hollee desktop menu.
- **Windows file import:** add files to Hollee Library from Windows Explorer using the Hollee right-click or Send To option.
- **Desktop updates:** installed apps check this release page and show an in-app update button when a new approved version is available.
