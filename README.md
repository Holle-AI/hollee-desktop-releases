# Hollee Desktop

![Hollee Desktop](assets/banner.jpg)

**Fall in love with work again.**

Hollee Desktop brings Hollee into a focused Windows app, giving teams a cleaner place to work with the live Hollee platform outside the browser.

## Download

Download the latest Windows installer from this repository's Releases page:

```txt
Hollee-Desktop-Setup-x.y.z.exe
```

The setup app installs Hollee Desktop and keeps it ready for future updates.

## Updates

Hollee Desktop checks this public releases repository for approved production updates. When an update is available, the app shows an update button in the title bar so users can download, restart, and move to the latest version.

These files are required for updates and should stay attached to each release:

```txt
Hollee-Desktop-Update-x.y.z.exe
Hollee-Desktop-Update-x.y.z.exe.blockmap
latest.yml
```
